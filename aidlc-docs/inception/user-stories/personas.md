# User Personas

## Persona 1: 김민수 (Customer - 고객)

**역할**: 레스토랑 방문 고객

**주요 목표**:
- 빠르고 편리하게 메뉴를 주문하고 싶음
- 대기 시간 없이 즉시 주문하고 싶음
- 주문 내역을 쉽게 확인하고 싶음

**관련 User Stories**: US-C-001 ~ US-C-008

---

## Persona 2: 박지영 (Admin - 매장 관리자)

**역할**: 레스토랑 매장 운영자/관리자

**주요 목표**:
- 실시간으로 들어오는 주문을 효율적으로 관리하고 싶음
- 테이블별 주문 상태를 한눈에 파악하고 싶음
- 메뉴를 쉽게 등록하고 수정하고 싶음
- 과거 주문 내역을 조회하고 분석하고 싶음

**관련 User Stories**: US-A-001 ~ US-A-009, US-NFR-001 ~ US-NFR-003
